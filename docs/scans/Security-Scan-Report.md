# Security Scan Results (OWASP ZAP)

The full, detailed security scan reports are no longer stored directly in this repository to maintain optimal performance and version control history.

## Where to find the latest report

To view the most recent automated security analysis:

1. Navigate to the **[Actions]** tab of this GitHub repository.
2. Select the most recent workflow run (usually named `Generate Zap Scan`).
3. Scroll down to the **Step Summary** section for a high-level overview of findings.
4. For the full technical report (HTML/JSON), check the **Artifacts** section at the bottom of the run page.

## Retention Policy

* **Step Summaries:** Available for **90 days** following the scan.
* **Full Artifacts:** Available for **90 days**.
